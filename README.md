# Modern Notes - Airplane

An interactive, modern note-taking application that transforms your notes into paper airplanes.

## Features
- **Modern UI**: Soft gradients (off-white, yellow, sky blue).
- **Interactive Animation**: Watch your note fold and fly away when you click "Send".
- **Responsive**: Works on screens of all sizes.

## How to use
1. Open `index.html` in any modern web browser.
2. Type your note in the card.
3. Click the "Send Note" button to see the magic happen.

## Technologies
- HTML5
- CSS3 (Custom properties, Keyframe animations, Flexbox)
- Vanilla JavaScript
